# Images Folder

This folder is designated for figures extracted from the source PDF or created for the chapter.

## Required Images

The following images are needed for complete documentation:

1. **tetracycline-structures.png** - Chemical structures of various tetracycline antibiotics (Figure 25.1 from source)
   - Should show the four-benzene ring core structure
   - Illustrate substitution positions at C-5, C-6, and C-7
   - Compare structures of tetracycline, doxycycline, minocycline, tigecycline, eravacycline, and omadacycline

2. **mechanism-of-action.png** - Diagram showing tetracycline binding to 30S ribosomal subunit
   - Should illustrate blocking of aminoacyl-tRNA at A site

3. **resistance-mechanisms.png** - Overview of tetracycline resistance mechanisms
   - Efflux pumps
   - Ribosomal protection proteins
   - Enzymatic inactivation

## Notes on Image Extraction

- Original figures should be extracted from the PDF using tools like `pdfimages` or Adobe Acrobat
- Ensure images are saved at sufficient resolution (300 DPI minimum)
- Use PNG format for diagrams, JPG for photographs
- Maintain proper attribution and copyright notices
